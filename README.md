# Farmer Stress Detection System

## 📌 Overview
This project is a Machine Learning-based system that detects stress levels in farmers using physiological, voice, and contextual features.

## 🚀 Features
- Predicts stress levels (Low, Medium, High)
- Uses Random Forest Classifier
- Provides analysis based on feature importance
- Includes ROC Curve and Confusion Matrix

## 🧠 Model Used
- Random Forest Classifier

## 🛠 Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## 📊 Input Features
- Heart Rate
- HRV
- EDA
- Skin Temperature
- Pitch
- Energy
- Work Hours
- Financial Stress
- Weather Stress
